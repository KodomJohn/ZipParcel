ZipParcel Logistics Analytics.
A end-to-end data analytics project simulating a real parcel delivery company.

- Project Overview
ZipParcel tracks customers, drivers, vehicles, routes, and deliveries across New York City. This project demonstrates the full data analytics workflow — from database design to business insights and visual dashboards.

- Tools Used
MySQL — database design & querying
MySQL Workbench — query execution & data export
Microsoft Excel — dashboard & data visualization

- Key Insights
Manhattan leads with the highest customer concentration
